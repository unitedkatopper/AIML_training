# AM Report Notes

Use these sections in the project report:
- Dataset Description: data/attempts.csv is generated from user assessment responses.
- Data Preprocessing: missing values are dropped during training; scores are converted to numeric columns.
- EDA: run `python eda.py` to generate graphs in results/eda/.
- Models: Random Forest Regressor, Random Forest Classifier, KMeans Clustering.
- Evaluation: run `python train.py` to generate results/model_metrics.csv.
